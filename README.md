# Deep Learning Practical Works (TPs)

This repository contains a series of Jupyter notebooks and complementary PDF documents covering various Deep Learning topics. Each "TP" (Travaux Pratiques) focuses on a specific aspect of modern neural networks.

## Repository Contents

### Jupyter Notebooks
- **TP1: Introduction to Neural Networks** (`TP1_Intro_to_NNs.ipynb`)
- **TP2: Convolutional Neural Networks (CNNs)** (`TP2_CNNs.ipynb`)
- **TP3: Vision Transformers** (`TP3_Vision_Transformers.ipynb`)
- **TP4: Transfer Learning** (`TP4_Transfer_Learning.ipynb`)
- **TP5: Visualizing Neural Networks** (`TP5_Visualizing_Neural_Networks.ipynb`)
- **TP6: Domain Adaptation** (`TP6_Domain_Adaptation.ipynb`)
- **TP7: Generative Adversarial Networks (GANs)** (`TP7_GAN.ipynb`)
- **TP8: Diffusion Models & Flow Matching** (`TP8_DiffusionModels_flowMatching.ipynb`)
- **TP9: Bayesian Neural Networks** (`TP9_Bayesian_neural_Network.ipynb`)
- **TP10: Uncertainty in Deep Learning** (`TP10_Uncertainty.ipynb`)

### Question & Answer Documents (PDFs)
The repository also includes several PDF files providing theoretical background and answers to key questions for each topic:
1.  **Intro to Neural Networks** (`Questions_Answers1_Intro_Neural_Network.pdf`)
2.  **CNNs** (`Questions_Answers2_CNN.pdf`)
3.  **Transfer Learning** (`Questions_Answers3_Transfer_Learning.pdf`)
4.  **Visualizing Neural Networks** (`Questions_Answers4_Visualizing_Neural_Networks.pdf`)
5.  **Domain Adaptation** (`Questions_Answers5_domain_adaptation.pdf`)
6.  **GANs** (`Questions_Answers6_GANs.pdf`)
7.  **Diffusion Models & Flow Matching** (`Questions_Answers7_DiffusionModels_Flow_matching.pdf`)

## Getting Started
To run the notebooks, ensure you have a Python environment with common Deep Learning libraries installed (such as PyTorch or TensorFlow, NumPy, and Matplotlib).

```bash
jupyter notebook
```
